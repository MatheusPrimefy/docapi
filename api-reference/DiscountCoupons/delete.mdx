* Request type: **\[DELETE]**

```json
https://api.premepay.com/stores/:id/vouchers/:id-ou-code
```


You have the possibility to permanently delete a Discount Coupon, as long as it is not associated with a Payment Order. If it is associated, the status will change to Inactive and it will not be possible to locate or use this Discount Coupon in the future.
