* Request type: **\[GET]**

```json
https://api.premepay.com/v1/stores/:id/vouchers/:id-ou-code
```

### **Responses**

If your request is successful, you will receive the return of the Voucher object.

* 200: OK

```json
  {
      "id": 11,
      "code": "PREME15",
      "mode": "Fixed",
      "amount": 10.00,
      "expirationDate": "2022-06-23T00:00:00",
      "paymentTypes": [
          1,
          3
      ],
      "status": "Inactive",
      "createdOn": "2022-05-20T15:11:47"
  }
  ```
