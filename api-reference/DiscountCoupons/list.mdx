Obtaining registered Discount Coupons.

* Request type: **\[GET]**

```json
https://api.premepay.com/v1/stores/:id/vouchers/
```

Would you like to list discount coupons by Business?

Use the endpoint:

```json
    https://api.premepay.com/v1/businesses/:id/vouchers/
```
