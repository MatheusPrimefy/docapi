---
title: 'Add new images to the product'
openapi: POST 'https://api.premepay.com/v1/stores/:id/products/:id/images'
---

* Request type: **\[POST]**

```json
https://api.premepay.com/v1/stores/:id/products/:id/images
```


Following the route above, you can register up to five images at once. The encodedFile property needs the image to be encoded in base64 format.

### Request example

This is an example of what your request to authenticate might look like.

- Body
    
    ```json
    {
    	"images": [
    		{
    			"encodedFile": "data:image/jpeg;base64",
    			"description": "some description",
    		},
    		{
    			"encodedFile": "data:image/jpeg;base64",
    			"description": "some description",
    		},
    		{
    			"encodedFile": "data:image/jpeg;base64",
    			"description": "some description",
    		},
    		{
    			"encodedFile": "data:image/jpeg;base64",
    			"description": "some description",
    		}
    	]
    }
    ```
