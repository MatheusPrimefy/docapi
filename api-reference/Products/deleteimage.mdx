* Request type: **\[DELETE]**

```json
https://api.premepay.com/v1/stores/:id/products/:id
```

You have the possibility to delete images associated with the Products whenever you wish. This action only removes the image and does not change or remove the Product.

>
⚠️ **Attention:** This action is irreversible.
