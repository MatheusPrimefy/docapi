openapi: GET 'https://api.premepay.com/v1/stores/:id/products/:id/images'

* Request type: **\[GET]**

```json
https://api.premepay.com/v1/stores/:id/products/:id/images
```

* Would you like to get specific product by Business?Use the endpoint

```json
https://api.premepay.com/businesses/:id/products/:id
```
