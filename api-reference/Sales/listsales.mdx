* Request type: **\[GET]**

```json
https://api.premepay.com/v1/stores/:id/orders
```


Would you like to list sales as Business?
 Use the endpoint:
   ```json
    `https://api.premepay.com/v1/businesses/:id/orders`
    ```

Through the above routes you will be able to view a list of all payment orders. Check below the possible responses of the Primefy API for this request.
