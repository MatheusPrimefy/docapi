---
title: 'Disassociate a product from the plan'
openapi: DELETE 'https://api.premepay.com/v1/stores/:id/plans/:id'
---

* Request type: **\[DELETE]**

```json
https://api.premepay.com/v1/stores/:id/plans/:id
```
You have the possibility to disassociate a Product from a subscription plan, provided that no sales have been generated using the Plan to which this Product is related.

>⚠️ **Attention:** This action is irreversible.


### Responses

- 204: No Content
    
    This is the answer for cases where your request is successful.
    
- 400: Bad Request
    
    For scenarios where the Product has already been associated with a Sale
    
    ```json
    {
        "message": "the product :id is already associated with one or more orders and cannot be deleted"
    }
    ```
    
    For scenarios where the Product is not found
    
    ```json
    {
        "message": "product not found"
    }
    ```
    
- 401: Unauthorized
    
    This is the response for cases where the request was unsuccessful because it lacks valid authentication credentials for the target resource.
    If needed, you can review our [authentication process](https://www.notion.so/Authentication-c941a0cb8e2647b187d87fcf8a314b37?pvs=21).****

    ```json
        "type": "https://httpstatuses.com/401",
        "title": "Unauthorized",
        "status": 401,
        "detail": null,
        "instance": null,
        "extensions": {}
    }
    ```
    
- 403: Forbidden
