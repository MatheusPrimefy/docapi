title: 'List plans'
openapi: POST 'https://api.premepay.com/v1/stores/:id/plans'
---

* Request type: **\[GET]**

```json
https://api.premepay.com/v1/stores/:id/plans
```

Through the above route you will receive a list of all your registered subscription plans. This is a preview list.

Check out the possible responses from the Primefy API to this request below.
