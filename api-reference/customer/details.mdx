title: "List customer details"
openapi: " GET https://api.premepay.com/v1/stores/:id/customers/:identification-number"
---

* Request type: **\[GET]**

```json
https://api.premepay.com/v1/stores/:id/customers/:identification-number
```

Through the mentioned route above, it is possible to list even more details of a consumer, using their CPF. The response will provide information ranging from registration details to sales, subscriptions, refunds, etc.
